# WEPR Growth Skills

Three practical Agent Skills for public relations, digital growth, and paid media. The repository focuses on diagnosis, execution, measurement, and risk boundaries rather than generic advice or guaranteed results.

## Skills

- `diagnose-pr-crisis` — Public-relations incident diagnosis, crisis statements, media preparation, stakeholder response, and reputation recovery.
- `audit-digital-growth` — Analytics, funnels, attribution, conversion, CRM, retention, dashboards, and growth roadmaps.
- `plan-paid-media` — Search, Douyin, Xiaohongshu, WeChat Channels, account structure, testing, optimization, and multi-platform planning.

## Install

Copy one or more folders under `skills/` into your Agent skills directory, such as `~/.codex/skills/` or `~/.agents/skills/`.

## Design principles

- Execution before slogans
- Evidence before certainty
- Business outcomes before vanity metrics
- Current official platform rules before remembered formulas
- Honest boundaries before result promises
