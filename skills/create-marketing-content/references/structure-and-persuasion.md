# Structure and Persuasion

## Choose the structure by reader job

| Reader job | Structure |
| --- | --- |
| Learn a method | Problem → mechanism → steps → example → checklist |
| Diagnose a failure | Symptom → evidence → competing causes → test → action |
| Make a choice | Options → criteria → trade-offs → recommendation → exception |
| Understand a case | Context → decision → execution → evidence → limits → transfer |
| Reconsider a belief | Common view → contradiction → mechanism → evidence → implication |

## Persuasion hierarchy

Prefer, in order: direct evidence, observable demonstration, relevant example, causal mechanism, credible attribution, and clearly labeled opinion. Emotion can support attention and memory but must not replace proof.

## Useful tension

Create tension through a real gap between goal and constraint, common action and hidden consequence, short-term metric and long-term outcome, or two legitimate trade-offs. Never manufacture personal shame, status panic, or false scarcity.
