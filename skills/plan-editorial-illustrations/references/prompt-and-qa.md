# Illustration Prompt and QA

## Prompt order

Specify: format → background → subject → action → composition → environment → labels → palette → line/texture → whitespace → exclusions.

Keep instructions concrete. Replace “show growth” with an observable scene such as a character moving verified blocks up a staircase while discarded shortcuts fall away.

## Quality checklist

- One dominant idea is readable at thumbnail size.
- The main subject occupies roughly 40–60% of the active area when appropriate.
- Whitespace supports hierarchy rather than appearing unfinished.
- Labels are few, short, accurate, and necessary.
- Names, numbers, quotations, and brand terms match the source.
- Character proportions, face logic, palette, and line treatment remain consistent.
- No cropped hands, accidental extra limbs, duplicated objects, or unreadable text.
- The result does not resemble a dense presentation slide, generic stock graphic, or copied signature artwork.

If text accuracy is critical and the image model cannot reproduce it reliably, generate the illustration without text and add verified typography in a deterministic layout tool.
